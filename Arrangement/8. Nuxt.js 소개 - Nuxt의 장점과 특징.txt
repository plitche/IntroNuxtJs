	8. Nuxt.js 소개 - Nuxt의 장점과 특징
		* 장점
			- 검색 엔진 최적화
			- 초기 프로젝트 설정 비용 감소와 생산성 향상
				> ESLint, Prettier
				> 라우터, 스토어 등의 라이브러리 설치 및 설정 파일 필요X
				> 파일 기반의 라우팅 방식. 설정 파일 자동 생성
			- 페이지 로딩 속도와 사용자 경험 향상
				> 브라우저가 하는 일을 서버에 나눠준다.
				> 모르면 지나칠 수 있는 코드 스프리팅이 기본으로 설정
				
		* 특징
			- 서버 사이드 렌더링
			- 규격화된 폴더 구조(layout, store, middleware, plugins 등)
			- pages 폴더 기반의 자동 라우팅 설정
			- 코드 스프리팅
			- 비동기 데이터 요청 속성
			- 웹팩을 비롯한 기타 설정
